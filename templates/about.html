NutriGaze

AI Based Fruit & Vegetable Freshness Classification

Project Overview

NutriGaze is an AI-powered web application that detects whether fruits and vegetables are fresh or rotten using Deep Learning and Image Classification. Users can upload an image and instantly receive freshness prediction results.

Technical Skills Used

  • Python
  • Flask Web Framework
  • TensorFlow / Keras
  • Transfer Learning (MobileNetV2)
  • HTML, CSS, JavaScript

System Workflow

User Uploads Image
↓
Flask Server Receives Image
↓
Image Preprocessing
↓
CNN Model Prediction
↓
Freshness Result Displayed
